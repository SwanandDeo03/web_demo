# TechStore - Electronics Shopping Website

A modern, fully functional e-commerce website for shopping electronics items built with HTML, CSS, and JavaScript.

## Features

- **Homepage** with hero section and featured products
- **Product Listing Page** with filtering and sorting capabilities
- **Product Detail Page** with quantity selector
- **Shopping Cart** with add/remove/update functionality
- **Responsive Design** that works on all devices
- **Category Filtering** by product type
- **Price Sorting** (low to high, high to low)
- **Local Storage** for cart persistence
- **Modern UI/UX** with smooth animations and transitions

## Pages

1. **index.html** - Homepage with featured products and category navigation
2. **products.html** - All products with filtering and sorting options
3. **product-detail.html** - Individual product details page
4. **cart.html** - Shopping cart with order summary

## How to Use

1. Open `index.html` in your web browser
2. Browse products by clicking on categories or navigating to the Products page
3. Click on any product to view details
4. Add products to cart
5. View and manage your cart
6. Proceed to checkout (demo mode)
